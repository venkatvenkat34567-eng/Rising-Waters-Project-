# 🌊 Rising Waters Project

## 🚀 Machine Learning Based Flood Prediction System

### 📖 Project Overview

Rising Waters is a Machine Learning-based Flood Prediction System developed using Python and Flask. The application predicts the possibility of floods based on environmental parameters using a trained Machine Learning model. It provides a simple web interface where users can enter input values and instantly receive flood prediction results.

---

## 🎯 Objectives

- Predict flood risk using Machine Learning.
- Help users estimate flood possibility from environmental data.
- Provide a simple and interactive web application.
- Improve awareness and disaster preparedness.

---

## ✨ Features

- 🌊 Flood Prediction using Machine Learning
- 🖥️ User-Friendly Web Interface

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/venkatvenkat34567-eng/Rising-Waters-Project-.git
```

### Go to the project folder

```bash
cd Rising_Waters_Project
```

### Install required packages

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

### Open your browser

```
http://127.0.0.1:5000
```
